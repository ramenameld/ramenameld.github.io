How Do We Work

We create a new jewellery design by matching ourselves and your intelligence, which makes that design very attractive and best in the world of jewellery designs.

Concept & Designing

We start the process in the conceptual phase. Several sketches are created using different designs. These sketches also determine the jewellery's weight, scale and size.


Workshop Work

After creating concept, our workers start work on making jewellery's base using gold metal. It takes upto 4-5 days.


Ghat/Base Ready

Then, jewellery's ghat/base is ready to use for next process. Each piece of jewellery is created separately. After ghat/base is determined, the jewellery moves to the final manufacturing process.




Gems Setting

The key part of every jewellery is its gems. The gems makes jewellery more beautiful. It adds beauty to the jewellery.
